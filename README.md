## 적용 기술 및 구현 기능

### 적용 기술

> - Front-End : React.js, sass, slick, react-modal
> - Back-End : node.js, express, jwt, Bcrypt, My SQL
> - Common : RESTful API, git, github



### 구현 기능

#### 공통

- 일반 회원가입 / 로그인
- 어쩌구


#### 메인페이지

- 추천 상품, 오늘의 상품, 베스트 상품 모두 각각 api 호출하여 구현
- 저쩌구
